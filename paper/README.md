# Paper And Supplement

This directory is reserved for paper drafts, supplements, reviewer appendices,
and historical research context.

Current contents:

- `legacy-prototype-fragment.md`: preserved root-level prototype fragment. It is
  historical context, not executable package code.

Suggested future additions:

- Method appendix describing DRR assumptions and notation.
- Reproducibility appendix referencing `results/reproduction/`.
- Validation appendix describing synthetic benchmarks and known limitations.
- Figure sources for architecture and workflow diagrams.

Do not add invented benchmarks, fabricated citations, or claims not supported by
checked-in artifacts.
